# Template

## Running locally

Setup WordPress locally.

```
yarn run setup
```

Install any dependencies.

```
yarn install
```

To run the site locally, [`valet`](https://laravel.com/docs/8.x/valet) can be used from the root directory.

```
valet link template
```

Open the site with Valet.

```
valet open
```
